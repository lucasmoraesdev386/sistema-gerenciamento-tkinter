# Sistema de Gerenciamento com Interface Gráfica

Um sistema simples e funcional de cadastro de clientes desenvolvido em **Python com Tkinter** e banco de dados **SQLite**.

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/Tkinter-3776AB?style=for-the-badge)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=for-the-badge&logo=sqlite&logoColor=white)

## Funcionalidades

- Cadastro de clientes (Nome, Email e Telefone)
- Interface gráfica intuitiva com Tkinter
- Banco de dados SQLite (persistência de dados)
- Visualização dos registros em tabela
- Data e hora automática de cadastro
- Validação de campos obrigatórios

## Tecnologias Utilizadas

- **Python 3**
- **Tkinter** (Interface Gráfica)
- **SQLite3** (Banco de Dados)
- **ttk** (Componentes visuais aprimorados)
